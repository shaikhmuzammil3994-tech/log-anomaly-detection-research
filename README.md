## 📌 Overview

This project presents a hybrid deep learning framework combining CNN and Transformer architectures with structural feature engineering for log anomaly detection.

It improves performance on benchmark datasets such as HDFS and BGL.

---

## 🚀 Features

- CNN for local pattern extraction  
- Transformer for sequence learning  
- Structural feature engineering  
- ROC-AUC evaluation  
- Streamlit demo app  

---

## 📊 Results

- ROC-AUC: **0.91**
- Outperforms:
  - TF-IDF
  - Isolation Forest
  - LSTM
  - Transformer-only models
